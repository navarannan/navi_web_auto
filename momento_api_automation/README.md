# rSpec API framweork for AirVisual

rSpec API framweork to test AirVisual Rest APIs

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for testing purposes.

### Prerequisites

Softwares needed to run the tests locally

```
Rubymine IDE
Ruby v 2.5.5 for Windows
Git bash for Windows
```


## Running the tests

Clone the repository on Git. On git bash execute the following


```
cd momento_api_automation
gem install rake bundler
bundle install
rspec spec/airvisual_api_tests.rb
```
